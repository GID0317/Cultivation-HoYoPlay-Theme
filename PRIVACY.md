# Privacy Policy

**Last updated: July 7, 2026**

## Overview

Cultivation HoYoPlay Theme is a visual theme for the Cultivation launcher.

This theme does not collect, sell, or transmit personal information to the project maintainer. It runs inside the Cultivation aplication and is intended to change the aplication appearance and related UI behavior.

## Local Data

The theme stores small local preferences in the launcher WebView storage so the UI can remember user choices. Current examples include:

- whether built-in backgrounds are enabled
- whether HoYoPlay background rotation is enabled
- the selected HoYoPlay background circle
- per-circle animated background play/pause intent
- cached background URL lists and timestamps used to avoid repeated background lookups

These values are stored locally on the user's device and are used only for theme behavior.

## Network Requests

The theme loads some visual assets and background media from GitHub raw content URLs. It may also request GitHub release metadata to show version/update information for Cultivation or this theme.

Network requests currently include GitHub-hosted assets, background images or videos, and GitHub release API endpoints. GitHub and any other referenced third-party services may receive standard request information such as IP address, user agent, request URL, and request time according to their own privacy policies.

## Analytics and Tracking

This theme does not include analytics, telemetry, advertising trackers, or user behavior tracking.

## Third-Party Services

This project may link to or load resources from GitHub, HoYoVerse/HoYoPlay, Cultivation, Grasscutter, or related resources. Those services are governed by their own privacy policies.

## Changes to This Policy

Any updates to this privacy policy will be reflected in the theme GitHub repository.

## Contact

For questions or concerns, please open an issue in this GitHub repository.
